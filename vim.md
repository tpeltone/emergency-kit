# Vim

## Useful links

[Interactiv Vim Tutorial](https://www.openvim.com/)

### Useful Commands

 ```bash
# Open file in vim
vim myfile.sh

# force quit
esc : q!

# Write and quit file
esc : wq

# Delete from cursor to end of word
de

# Delete everything in a file
gg shift+VGD

# Highlight a row
shift+V

# Copy a row
shift+Y

# Paste
p

# Undo
u

# Redo
ctrl+R
 ```
