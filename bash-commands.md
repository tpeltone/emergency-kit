# Bash commands

Create a folder:

```bash
mkdir myfolder
```

Change directory:

```bash
cd myfolder
```

List files and folders:

```bash
ls -a
ls -l
ls -F
ls -laF
ls -f (aF)
```

SSH from a bash prompt to a remote server:

```bash
ssh server-name.x.y.z
```

Run bash as super user:

```bash
sudo su
```

Run bash as a specifik user that exists:

```bash
sudo su - <the user>
```
